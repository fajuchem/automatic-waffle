## Descrição

Desafio da WaProject

## Instalação

```bash
$ npm install
```

## Rodar a aplicação

```bash
# Desenvolvimento
$ npm run start:dev

# Produção
$ npm run start:prod
```

## Rodar a aplicação no docker

```bash
# Desenvolvimento
$ docker-compose up dev
```

## Testes

```bash
# Teste unitário
$ npm run test

# Teste cobertura
$ npm run test:cov
```
